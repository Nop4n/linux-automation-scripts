Ansible vs Bash Scripts for a small team (1-5 servers):

Ansible:
+ Declarative, idempotent
+ Huge module library
- Requires Python on all targets
- YAML debugging is painful
- Learning curve for non-developers
- Overhead for simple tasks

Structured Bash:
+ Runs everywhere Linux does
+ Zero dependencies
+ Easy to understand and modify
+ Fast execution
- No built-in idempotency
- Error handling is manual

For small teams: bash scripts with proper structure beat Ansible in speed and simplicity.
